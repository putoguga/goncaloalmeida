  • Baloon-in-line

Baloon-in-line

This project is a based game inspired by the classic Four-in-line, where two players compete to align four balloons in a row—horizontally, vertically, or diagonally. The game features a clear ASCII-style board and supports two-color pieces, making it easy to follow the progress. It's a fun and nostalgic implementation that brings simple gaming to the command line!

Boat Radar

This project is a based simulation that tracks the positions of boats on a grid. Users can view the initial and final positions of each boat, check individual boat coordinates, and list which boats have moved. This visual representation helps simulate a simple radar system, making it easier to monitor maritime traffic in a controlled environment.
